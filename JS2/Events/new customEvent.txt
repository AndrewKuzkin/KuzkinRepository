new CustomEvent - это конструктор JavaScript, который создает новое событие пользовательского типа. Он позволяет создавать события, которые не являются стандартными для браузера, а представляют собой пользовательские события, определяемые разработчиком.

Синтаксис:
var event = new CustomEvent(eventType, options);
- eventType: строка, представляющая тип события. Это может быть любая строка, которую вы хотите использовать для вашего пользовательского события.
- options: объект со следующими опциональными свойствами:
  - bubbles: логическое значение, указывающее, должно ли событие всплывать (по умолчанию false).
  - cancelable: логическое значение, указывающее, можно ли отменить событие (по умолчанию false).
  - detail: любое значение, которое вы хотите передать в событие.

Пример:
var event = new CustomEvent('customEvent', {
  bubbles: true,
  cancelable: true,
  detail: { message: 'Hello' }
});


Этот код создает новое пользовательское событие с именем 'customEvent', которое может всплывать и быть отменено, и передает объект { message: 'Hello' } в качестве дополнительной информации через свойство detail.
