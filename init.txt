1. Привязать свою  директорию к репозиторию
git init [<твоя директорияy>]
2. Добавить удаленный репозиторий, к которому вы хотите привязать свою локальную директорию, с помощью команды:‍
git remote add origin <URL_удаленного_репозитория>
Здесь <URL_удаленного_репозитория> - это URL вашего удаленного репозитория. Например:‍
git remote add origin https://github.com/username/repository.git
3. Добавить файлы в индекс и сделать коммит, чтобы зафиксировать текущее состояние вашей директории:‍
git add .
   git commit -m "Initial commit"
   ‍
4. Отправить изменения на удаленный репозиторий с помощью команды:‍
git push -u origin master
Эта команда отправит ваш локальный коммит на ветку master удаленного репозитория.‍
‍Теперь ваша локальная директория привязана к удаленному репозиторию. Вы можете вносить изменения в свои файлы, делать коммиты и отправлять их на удаленный репозиторий с помощью команд git add, git commit и git push.‍
