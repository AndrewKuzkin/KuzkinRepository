###1. 
Демонстрационная  версия интерфейса обучающей системы и документация по нейнаходится в кпапке AskUI
###2.
Необходимо распаковать архив AskUI.zip и запустить файл index.htm

###3. 
Интерфейс обучающей системы ASK использует фреймы, поэтому для навигации по ним  со скринридером нужно использовать специальные клавиатурные команды(для NVDA нужно жать "m")
Основная информация отображается в main-фрейме, поэтому при работе со скринредероми после любого действия  нужно прочитать содержимое этого фрейма.
###4. 
Невизуальная доступность протестирована в OS Windoes 10 со скринридером NVDA и браузерами Firefox, Google Chrome, Яндекс-браузером, Microsoft Age и atom.
