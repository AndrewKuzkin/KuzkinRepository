Для создания серверного скрипта на JavaScript, который будет обрабатывать текстовый ввод и выводить его на страницу, можно использовать Node.js в сочетании с простым веб-сервером. Вот пример кода:

1. Установите Node.js, если у вас его еще нет.
2. Создайте новую директорию для проекта и перейдите в нее через терминал.
3. Установите пакет http для создания веб-сервера и fs для работы с файловой системой:

npm install http fs


4. Создайте файл server.js и добавьте следующий код:

const http = require('http');
const fs = require('fs');

const server = http.createServer((req, res) => {
    if (req.method === 'POST' && req.url === '/') {
        let body = '';
        req.on('data', chunk => {
            body += chunk.toString();
        });
        req.on('end', () => {
            res.writeHead(200, { 'Content-Type': 'text/html' });
            res.end(<h1>Вы ввели следующий текст:</h1><p>${body}</p>);
        });
    } else {
        res.writeHead(404, { 'Content-Type': 'text/plain' });
        res.end('Страница не найдена');
    }
});

const PORT = 3000;
server.listen(PORT, () => {
    console.log(Сервер запущен на порту ${PORT});
});


5. Запустите сервер, выполнив в терминале:

node server.js


Теперь ваш сервер будет прослушивать POST запросы на порту 3000. Вы можете отправлять POST запросы на http://localhost:3000/ с текстовым телом, и сервер будет выводить этот текст на страницу.

Например, вы можете использовать инструменты для отправки POST запросов, такие как curl или Postman, чтобы отправить текстовые данные на ваш сервер и увидеть результат.

Этот код представляет базовый пример серверного скрипта на JavaScript для обработки текстового ввода и вывода его на страницу. Вы можете доработать его, добавив дополнительную логику обработки данных или интерфейс для взаимодействия с сервером через браузер.
